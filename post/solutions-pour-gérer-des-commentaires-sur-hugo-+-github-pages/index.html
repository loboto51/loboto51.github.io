Solutions pour gérer des commentaires sur Hugo + Github Pages

10/05/2021

3 minutes


Veille sur les systèmes de commentaire intégrables à Hugo + Github Pages.

La doc Hugo de référence : https://gohugo.io/content-management/comments/

Discussion sur le forum Hugo : lternative-to-disqus-needed-more-than-ever

Supports possibles selon la doc + analyse perso (en gras ceux qui me semblent intéressants) :

  • Disqus
    • [-]Hébergement :
      • [-] Messages stockés chez Disqus
      • [-] Gratuit :
        • [-] beaucoup de pubs et de trackers (voir plus bas)
      • [-] Payant :
        • [-] Encore pas mal de trackers apparemment (voir plus bas)
    • [+] Commentaire invité : Oui
    • Décrit ici :
      • Vie privée, etc. : Gazoo.vrv - Replacing Disqus with Github Comments
      • Doc d’install : https://gohugo.io/content-management/comments/
  • Staticman
    • [+] Hébergement :
      • [+] Gratuit : Messages stockés chez Github (Github issues)
    • [+] Open source
    • [-] Commentaire invité : Non (compte Github)
    • Décrit ici :
      • Comparaison avec Utterances : BARTEKR - testing-utterances-comments
  • Talkyard
    • [/]Hébergement :
      • [/] Gratuit : Auto-hébergé
      • [/] Payant (2 euros par mois) : Messages stockés chez Talkyard
    • [+] Commentaire invité : Oui
    • [+] Open source
    • Décrit ici :
      • Talkyard (site officiel)
  • IntenseDebate
    • [-] Manque d’infos, interface un peu vieillotte
    • Décrit ici :
      • IntenseDebate (site officiel)
  • Graph Comment
    • [/]Hébergement :
      • [-] Messages stockés chez Graph Comment
      • [+] Gratuit jusqu’à 1 million de vues/mois
    • [+] Commentaire invité : Oui
    • Décrit ici :
      • GraphComment (site officiel)
  • Muut
    • [-]Hébergement :
      • [-] Messages stockés chez Muut
      • [-] Payant et tarif élevé : 16$/mois
    • Décrit ici :
      • Muut (site officiel)
  • Isso
    • [/]Hébergement :
      • [/] Gratuit : Auto-hébergé (Python + SQLite)
    • [/] Commentaire invité : Oui (mais pas de double authent à priori)
    • [+] Multilingue
    • Décrit ici :
      • Isso (site officiel)
  • Utterances
    • [+] Hébergement :
      • [+] Gratuit : Messages stockés chez Github (Github issues)
    • [-] Commentaire invité : Non (compte Github)
    • [+] Open source
    • [-] Seulement en anglais
    • Décrit ici :
      • Utterances (site officiel)
      • BARTEKR - testing-utterances-comments
  • Remark
    • [/]Hébergement :
      • [/] Gratuit : Auto-hébergé
    • [/] Commentaire invité : Non (mais beaucoup de méthodes : Google, Twitter, Facebook, Microsoft, GitHub ou Yandex)
    • [+] Open source
    • [-] Multilingue, mais pas de français
    • Décrit ici :
      • Github - remark42 (site officiel)
  • Commento
    • [-]Hébergement :
      • [-] Payant (10$/mois) : Messages stockés chez _Commento
    • Décrit ici :
      • Commento (site officiel)
  • Hyvor Talk
    • [-]Hébergement :
      • [-] Payant (5$/mois) : Messages stockés chez Hyvor
    • [+] Commentaire invité : Oui
    • Décrit ici :
      • Create a Free Blog Site Using GitHub Pages and Hugo#comments
      • https://talk.hyvor.com/

Autres pistes :

  • Twitter
    • [/] Hébergement :
      • [/] Gratuit : Messages stockés chez Twitter
    • [-] Commentaire invité : non (compte Twitter)
    • Décrit ici :
      • PÄKSTECH - I’m Switching to Twitter for My Blog Comments
      • How to use Twitter as comment section in your static blog and automatically create a tweet per post (Inspiré par le précédent)
  • Mastodon
    • [+] Hébergement :
      • [+] Gratuit : Messages stockés chez Mastodon
    • [-] Commentaire invité : Non (compte Mastodon)
    • [+] Open source
    • Décrit ici :
      • Using Mastodon for comments on a static blog
      • Comtodon - A minimal commenting system for static blogs using external Mastodon or API compatible server
      • mastodon-api-comments - Shortcode for Hugo to display the replies to a Fediverse-thread as blog comments.
  • (Autres) Github Issue API
    • [+] Hébergement :
      • [+] Gratuit : Messages stockés chez Github (Github issues)
    • [-] Commentaire invité : non (compte Github)
    • Décrit ici :
      • Gazoo.vrv - Replacing Disqus with Github Comments
  • Discourse
    • [/]Hébergement :

    • [-] Commentaire invité : Non
    • [+] Open source
  • Schnack
    • [/]Hébergement :

    • [-] Commentaire invité : Non (compte Github ou Twitter)
    • [+] Open source
    • [+] très léger
    • Décrit ici :
      • Say hello to Schnack.js: A new Disqus-like commenting drop-in for static websites
